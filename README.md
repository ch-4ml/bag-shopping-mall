# 온라인 남성 가방 쇼핑몰
학교 프로젝트로 진행한 온라인 남성 가방 쇼핑몰입니다.

## 프로젝트 개요

### 개발 기간
2017.09 ~ 2017.12

### 개발 언어
PHP

### OS
Linux

### DB
MySQL

### 개발 인원
1명

### 설명
고객전용 페이지와 관리자용 페이지가 나뉘어 있습니다.<br>
관리자는 회원관리, 상품관리, 주문관리 등을 할 수 있습니다.<br>
고객은 상품 검색, 구매, 장바구니추가 등을 할 수 있습니다.<br><br>
